# TweetX - Social Platform for Job Seekers and HR Professionals

Welcome to TweetX, a revolutionary social platform designed to transform the way job seekers connect with HR professionals and potential employers. This project is developed using Next.js, providing a dynamic user interface for a seamless user experience.

## Getting Started

To run the development server, use the following commands:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to explore the project. Start editing the page by modifying `app/page.js`. The page will auto-update as you make changes.

This project leverages [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to optimize and load Inter, a custom Google Font.

## Key Features

- **Job Seeker - HR Interaction:** TweetX facilitates direct communication between job seekers and HR professionals, providing a seamless platform for collaboration.

- **Real-time Updates:** Stay informed with real-time updates on job applications, interviews, and hiring processes.

- **Tech Twitter Integration:** Inspired by the concept of Tech Twitter, TweetX incorporates a social media touch to keep users updated on industry trends, job opportunities, and more.

## How It Works

1. **Apply for Jobs:** Job seekers can apply for positions and receive instant updates on their application status.

2. **Direct Communication:** Communicate directly with HR professionals to streamline the hiring process.

3. **Tech Twitter Feed:** Stay connected with the latest industry news and job trends through the integrated Tech Twitter feed.
